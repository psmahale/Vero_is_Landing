Examples

Python: Verify a file

from vero import VeroClient client = VeroClient(api_key="YOUR_KEY") resp = client.verify_document(file_path="./certificate.jpg", user_email="faculty@edu.in") print(resp)

cURL: Verify (base64)

curl -X POST https://api.vero.ai/v1/verify \ -H "X-API-Key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"file":"BASE64_DATA","user_email":"faculty@edu.in"}'
