Getting Started with VERO AI

Step 1: Get Your API Key

Sign up for a free account to receive your API key:

# Request API key curl -X POST https://api.vero.ai/v1/auth/register \ -H "Content-Type: application/json" \ -d '{"email": "your_email@domain.com", "institution": "Your University"}'

Step 2: Install the SDK

# Python SDK pip install vero-ai # Node.js SDK npm install vero-ai # Or use direct API calls

Step 3: Verify Your First Document

import vero client = vero.Client(api_key="your_api_key") result = client.verify( document="path/to/document.pdf", user_email="faculty@university.edu", claimed_score=15.0 ) print(f"Status: {result['status']}") print(f"Score: {result['score']}/100") print(f"Remarks: {result['remarks']}")
